# 📖 Guide Utilisateur — Site Artiste (React)

Ce site a été conçu pour présenter le travail d’un photographe professionnel de manière simple, fluide et immersive.  
Il a été optimisé conformément au cahier des charges pour un usage **desktop**.  
La version mobile est également responsive, bien que non exigée.

---

## 🌐 Navigation générale

Le site comporte 4 pages principales, accessibles via la barre de navigation ou le menu burger sur mobile :

- **Accueil**
- **Portfolio**
- **À propos**
- **Contact**

---

## 🏠 Page d’accueil

- Présentation texte placée en haut de page.
- Texte défilant dynamique.
- Catégories visuelles cliquables (« Portraits », « Voyage », etc.).
- Carrousel manuel d’images représentatives.
- Section newsletter en bas de page.

---

## 🖼️ Page Portfolio

- Affichage dynamique via API locale (JSON).
- Présentation en grille responsive.
- Clic sur une image/vidéo → ouverture en **plein écran (lightbox)**.
- Navigation fluide sans rechargement.

---

## 👤 Page À propos

- Biographie illustrée.
- Carte Google Maps du studio.
- Formulaire de contact rapide.
- Liens vers réseaux sociaux.

---

## 📬 Page Contact

- Adresse avec lien d’itinéraire.
- Email et téléphone cliquables.
- Carte Google Maps.
- Formulaire complet avec confirmation d’envoi.
- Inscription à la newsletter.

---

## ⚙️ Fonctionnalités techniques

- SPA développée avec **React + Vite**.
- Navigation fluide via **React Router**.
- Données chargées en **AJAX**.
- Formulaires contrôlés avec `useState`.
- Lightbox photo/vidéo officielle.
- Design responsive jusqu’à 320px.

---

## 💻 Compatibilité

- Navigateurs testés : Chrome, Firefox (versions récentes).
- Appareils : Desktop (principal), Tablette, Mobile.
- Optimisation validée pour desktop (cahier des charges).

---

## ⌨️ Raccourcis utiles

- **Échap** : fermer la lightbox.
- **← / →** : naviguer entre médias.

---

## ❓ FAQ rapide

- **Image qui ne s’ouvre pas ?** → Rechargez la page et réessayez.
- **Formulaire bloqué ?** → Vérifiez les champs obligatoires.
- **Affichage décalé ?** → Videz le cache du navigateur.

---

## 💡 Conseils pratiques

- Cliquez sur une image pour l’agrandir.
- Utilisez “Envoyer un message” pour contacter l’artiste.
- Cliquez sur le logo pour revenir à l’accueil.
- Sur mobile, utilisez l’icône burger pour naviguer.
